This plugin is developed to show links to your social networks.

== Description ==

ThemeNcode Social Follow Buttons is a lightweight plugin that comes with various options to display Link to your social networks. It comes with a shortcode as well as online shortcode generator. You can access the shortcode generator at: 

<a href="http://l.themencode.com/tsfbsc"> Shortcode Generator </a>

Configure your social networks on the generator & put the shortcode in any page or post of your wordpress site. Very easy to use. 

<a href="http://l.themencode.com/tsfbdemo">Live Demo</a>  |  <a href="http://l.themencode.com/tsfbdoc">Documentation</a>

<h3>Features :</h3>
* Very lightweight plugin
* 11 Social Networks including Facebook, Twitter, Youtube, Google Plus, Linkedin, Pinterest, Tumblr & more
* Choose any style of icon from 10 sets of icon.
* Easy to use shortcode generator

== Installation ==
Installing this plugin is pretty simple like all other wordpress plugins. Here are 2 ways to install :

<h3>Manual Install</h3>

* Download the zipped plugin from wordpress repository
* Extract the zipped folder
* Upload thenencode-social-follow-buttons to the /wp-content/plugins/ directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Go To <a href="http://l.themencode.com/tsfbsc">Shortcode Generator</a> to configure your social networks.
* Copy the generated shortcode & place it where you want the buttons to appear.

<h3>Automatic Install</h3>

* Go to Plugins-> Add New from your wordpress Dashboard
* Search ThemeNcode Social Follow Buttons
* From the results click on 'install now'
* Activate the plugin through the Plugins' menu in WordPress
* Go To <a href="http://l.themencode.com/tsfbsc">Shortcode Generator</a> to configure your social networks.
* Copy the generated shortcode & place it where you want the buttons to appear.
